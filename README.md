# Nest.js 学习计划

## 框架介绍

Nest 是构建高效且可扩展的 Node.js 服务器端应用程序的框架。它使用 TypeScript （保留与纯 JavaScript 的兼容性）构建，并结合了 OOP（面向对象编程）， FP（功能编程）和 FRP（功能反应编程）的元素。

### 主要特性

1. 完美支持 Typescript
2. 面向 AOP 编程
3. 支持 typeorm
4. Node.js 版的 spring
5. 构建微服务应用
