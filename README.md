## 框架介绍

Nest 是构建高效且可扩展的 Node.js 服务器端应用程序的框架。它使用 TypeScript （保留与纯 JavaScript 的兼容性）构建，并结合了 OOP（面向对象编程）， FP（函数式编程）和 FRP（函数式反应型编程）的元素。

### 主要特性

- 完美支持 Typescript
- 模块化开发
- 内置 IOC 容器
- 轻松编写 AOP 代码
- 支持 TypeORM
- 构建微服务应用
- ......
